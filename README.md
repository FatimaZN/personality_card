## 🌟 The Idea

You're going to build a Python program that asks a few fun questions, then prints a **beautiful "Personality Card"** — like a trading card or ID card made out of text and emojis.

The best part? It looks so cool that you'll **want to screenshot it and show your friends!** 📸

This is *your* card, made by *your* code. Let's make it awesome. 💪

---

## 🎯 What You'll Practice

Everything from Week 1 — all in one fun project:

- `input()` to ask questions
- **Type casting** (`int()` / `float()`) for the number facts
- **Variables** to store answers
- **Arithmetic operators** to calculate fun stats
- **Comparison / membership operators** to print fun `True/False` facts
- **f-strings** to design the card layout

> ⚠️ Stay inside Week 1 tools: **no `if` statements or loops** yet (those are Week 3). Just straight-line code. You've got everything you need!

---

## 🧱 Step-by-Step Build

### Day 1 — Collect the info

Ask the user (using `input()`) for at least:

- Name
- City
- Age *(remember to cast to `int`!)*
- A "superpower" (something fun they're good at — e.g. "Chai + Code")
- A lucky number *(cast to `int`)*

Print each answer back simply to test it works.

### Day 2 — Calculate fun stats

Use **operators** to create cool facts with f-strings:

- Age in **months** → `age * 12`
- How old they'll be in **2050** → `2050 - 2026 + age`
- Lucky number **× 7**
- A `True/False` fact using **membership**, e.g. *is the letter `"a"` in their name?* → `"a" in name`
- A `True/False` fact using **comparison**, e.g. *is their age over 18?* → `age > 18`

### Day 3 — Design the card 🎨

Arrange everything into a **framed card** using `print()` and f-strings. Add a border (using characters like `═ ║ ╔ ╗`) and emojis. Make it *yours*.
